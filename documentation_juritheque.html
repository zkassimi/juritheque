Vue d'ensemble

JuriThèque (nom de domaine : juritheque.com, nom interne du dépôt : lexbase) est une plateforme numérique de centralisation, de recherche et de vulgarisation du droit marocain.

Mission : Rendre accessibles les 7 400+ textes juridiques marocains (lois, dahirs, décrets, codes, arrêtés) en version bilingue français/arabe, avec résumés IA, assistant juridique intelligent, et contenus pédagogiques.

Architecture technique

JuriThèque repose sur une architecture JAMstack à 3 couches indépendantes :

🖥️ FRONTEND — React SPA
React 18 + Vite 5 + Tailwind CSS  |  Hébergé sur Hostinger (dist/ statique)
↕ REST API · Supabase JS SDK · clé anon publique
🗄️ BACKEND — Supabase Cloud
PostgreSQL + Auth JWT + Storage + PostgREST  |  Projet : bmargdbbcnhkrjeidmvh (Région EU)
↕ service_role key — pipeline uniquement, jamais public
⚙️ PIPELINE DE DONNÉES — Python local
28 scripts : crawl · scrape · extract · enrich · translate · build  |  LLM : OpenRouter → Gemini 2.5 Flash Lite  |  Dashboard admin : FastAPI port 8000

Flux de données

# Collecte
Sites officiels (11) ──► scraper.py / crawl_adala.py ──► PDFs locaux

# Extraction
PDFs locaux ──► extract.py ──► texte brut → Supabase laws

# Enrichissement IA (Gemini 2.5 Flash Lite via OpenRouter)
laws (sans résumé) ──► generate_adala_summaries.py ──► simple_summary_fr
laws (summary_fr)  ──► translate_summaries_ar.py   ──► simple_summary_ar

# Diffusion
Supabase PostgreSQL ──► REST API (anon) ──► Frontend React ──► juritheque.com

cd pipeline

pip install fastapi uvicorn

SUPABASE_SERVICE_KEY=[service_role key — droits admin complets]
OPENROUTER_KEY=[clé OpenRouter]

🧠 Enrichissement IA (OpenRouter)

💰 Ces scripts consomment des crédits OpenRouter. Vérifier le solde sur openrouter.ai/settings/credits avant de lancer. Tester d'abord avec --limit 10.
ScriptCommande typeCoût estimé
enrich.py--limit 200~$0.40 / 200 lois

Modes de generate_adala_summaries.py :

--mode null          # Lois sans résumé (nouveaux imports)
--mode placeholders  # Remplace les résumés génériques (6 896 à traiter)
--mode all           # Les deux combinés
--dry-run            # Simulation sans écriture en base
--limit N            # Limiter à N lois (test recommandé avec --limit 10)

Ordre recommandé pour la chaîne complète :

1. generate_adala_summaries.py --mode placeholders  → Résumés FR de qualité
2. translate_summaries_ar.py                        → Traduction FR→AR
3. Répéter jusqu'à 0 restants (visible dans la sidebar du dashboard)

Recommandations pour GitHub

  • 1. Ne jamais committer pipeline/.env, .env.local, pipeline/pdfs/
  • 2. Utiliser .env.example comme template (noms de variables, sans valeurs)
  • 3. Activer les Dependabot alerts GitHub sur le dépôt
  • 4. Utiliser GitHub Secrets pour les clés en CI/CD
  • 5. Protéger la branche main (require PR + review avant merge)
  • 6. Effectuer une rotation des clés API tous les 6 mois

Déploiement

🏠
Frontend actuel — Hostinger

Upload manuel du dossier dist/ via FTP ou cPanel File Manager. HTTPS Let's Encrypt. .htaccess pour le SPA routing.

npm run build
# → Upload dist/ vers public_html/
⚡
Frontend recommandé — Vercel/Netlify

Auto-deploy à chaque push sur main. HTTPS automatique, CDN mondial, rollback en 1 clic. Variables d'env dans le panel.

Avantage : zéro manipulation manuelle après configuration initiale.

🖥️
Pipeline actuel — Windows local

Scripts lancés manuellement ou via Windows Task Scheduler. Dépendance au PC local — risque si indisponible.

python -X utf8 pipeline/dashboard.py
☁️
Pipeline futur — VPS Linux

Cron jobs pour les tâches récurrentes. Service systemd pour le dashboard. Indépendant du PC local.

# Cron : vérif BO tous les jours à 2h
0 2 * * * python -X utf8 bo_monitor.py

Post-déploiement — Checklist SEO

  • Régénérer le sitemap : python -X utf8 pipeline/generate_sitemap.py
  • Soumettre à Google Search Console : https://juritheque.com/sitemap.xml
  • Vérifier le rendu SSR : https://juritheque.com/seo-preview.php?url=/loi/[id]
  • Tester les meta OG sur opengraph.xyz
  • Vérifier les CWV (Core Web Vitals) sur PageSpeed Insights

Projections & Roadmap
